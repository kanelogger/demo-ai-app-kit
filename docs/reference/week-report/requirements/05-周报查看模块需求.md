# 周报查看模块需求

## 1. 模块目标

供具备查看角色的用户查看已审核周报，范围受模板查看角色和部门范围约束。

## 2. 数据范围规则

- 仅展示已审核状态周报
- 仅展示模板已配置查看角色的数据
- 仅展示模板查看部门范围内的数据
- 不允许在该模块执行审核或修改

## 3. 查询条件

- 周报名称
- 所属部门
- 开始时间
- 结束时间

## 4. 列表字段

- 周报名称
- 填报人
- 所属部门
- 周报周期
- 审核时间
- 操作

## 5. 详情规则

- 从列表点击“查看”后，在右侧工作区打开新标签页，不使用弹窗
- 详情以只读表格化页面展示
- 需显示填报人、审核人、审核时间、审核意见

## 6. Demo 要求

- 使用不同账号切换时，需体现可查看范围差异
- 超级管理员默认可查看全部已审核周报

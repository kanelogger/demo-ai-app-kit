# 待我审核模块需求

## 1. 模块目标

用于直属领导查看待自己审核的周报，并完成审核通过或审核退回。

## 2. 列表规则

- 仅展示当前登录用户作为审核人的待审核周报
- 默认按提交时间倒序展示

## 3. 查询条件

- 周报名称
- 开始时间
- 结束时间

## 4. 列表字段

- 周报名称
- 填报人
- 填报部门
- 周报周期
- 提交时间
- 当前状态
- 操作

## 5. 审核页面规则

- 周报正文以只读表格方式展示
- 不显示可编辑控件
- 支持填写审核意见，审核意见非必填
- 支持审核通过和审核退回两个动作

## 6. 状态流转

- 待审核 -> 已审核
- 待审核 -> 已退回
- 已审核和已退回均不允许再次审核

## 7. Demo 要求

- 审核动作需即时刷新列表状态
- 审核意见需在“我填报的周报”中可见

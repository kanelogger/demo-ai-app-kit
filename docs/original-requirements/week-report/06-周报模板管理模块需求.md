# 周报模板管理模块需求

## 1. 模块目标

管理员配置周报模板、模板对象、数据项及其填报角色、查看角色和查看部门范围。

## 2. 模板基础信息

- 模板名称
- 模板编码
- 模板说明
- 状态
- 填报角色
- 查看角色
- 查看部门范围

## 3. 模板结构

- 一个模板包含多个数据对象
- 一个数据对象包含多个数据项
- 数据对象按显示顺序展示
- 数据项按显示顺序渲染为表格列

## 4. 数据项规则

- 数据类型：字符、数字、日期
- 控件类型：
  - 字符：单行文本、多行文本、下拉单选
  - 数字：数字输入
  - 日期：日期
- 下拉单选必须绑定某个数据字典类型

## 5. 维护能力

- 新增模板
- 编辑模板
- 启用/停用模板
- 新增/删除对象
- 新增/删除字段
- 填报角色、查看角色、查看部门范围均采用“文本框 + 按钮挑选”的方式维护
- 点击按钮弹出独立多选窗口，选择结果以逗号形式展示在文本框中

## 6. Demo 要求

- 模板修改后应直接影响待我填报页面渲染结构
- 需要保留模板版本字段，为后续结构变更控制预留能力

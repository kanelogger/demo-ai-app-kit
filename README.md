# demo-ai-app-kit

面向 AI Coding 场景的应用原型工具包，用于快速搭建可运行、可演示、可改造的 AI Web 应用。

## 项目定位

`demo-ai-app-kit` 不是一个固定业务系统，而是一套 AI Coding 和原型演示用的应用生产线。

它的目标是让开发者在拿到业务题目或演示需求后，不从空白项目开始，而是基于已有的后台管理模板、参考需求文档、设计文档、提示词和技能流程，快速完成：

- 业务方案细化
- 技术方案和页面设计
- 模板代码改造
- AI 工作流接入
- 本地联调
- 汇报材料准备

## 典型使用流程

1. 下载或安装本工具包。
2. 打开 OpenCode 或其他 AI Coding 工具。
3. 输入业务题目或演示需求。
4. 使用自定义技能追问和校验方案。
5. 基于模板调整页面、数据模型和业务流程。
6. 接入智能体工作流或大模型服务。
7. 完成本地测试和 10 分钟演示准备。
